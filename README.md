# GLODS-SI

**GLODS-SI** (Global and Local Optimization using Direct Search – Scale-Invariant)
is a scale-invariant reformulation of the GLODS framework for derivative-free,
bound-constrained global optimization.

This repository accompanies the manuscript:

> *Global and Local Optimization using Direct Search – A Scale-Invariant Approach (GLODS-SI)*  
> submitted to the *Journal of Global Optimization*.

## Contents

This repository will contain:
- A reference implementation of the GLODS-SI algorithm;
- The heterogeneous-scale benchmark instances used in the paper;
- Scripts to reproduce the numerical experiments.

## Status

The repository is being finalized and will be fully populated upon acceptance of
the manuscript. The current structure reflects the experimental pipeline
reported in the paper.
